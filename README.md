# Project Documentation

## Overview
This project is a personal website hosted on GitHub Pages. It is designed to showcase my portfolio, projects, and blog posts.

## Features
- Portfolio section to display work and projects
- Blog section for sharing thoughts and articles
- Responsive design
- Contact form for inquiries

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
To run this project locally:
1. Clone the repository: `git clone https://github.com/Don-Sanvura/Don-sanvura.github.io`
2. Open `index.html` in your web browser.

## Usage
Navigate through the sections using the menu. You can explore my projects, read blog posts, and contact me through the form.
